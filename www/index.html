Uncommons Antlib

Tools for building modular software with Apache Ant.

© 2008 Daniel W. Dyer - Licensed under the terms of the Apache Software Licence.

The Uncommons Antlib provides macro definitions, custom tasks and other tools to support the development of modular Java software using Apache Ant. It builds on the concepts and processes established in the development of several Ant-based projects. These include the Watchmaker Framework for Evolutionary Computation and Uncommons Maths open source projects.
