Uncommons Antlib

Tools for building modular software with Apache Ant.

The Uncommons Antlib provides macro definitions, custom tasks and other tools to support the development of modular Java software using Apache Ant. It builds on the concepts, processes and best practices established in the development of several Ant-based projects, particularly the Uncommons.org open source projects.

The guiding principle behind the design of the Uncommons Antlib is convention over configuration. Ant build scripts are greatly simplified by relying on consistent directory structures and naming conventions.

Uncommons Antlib requires APACHE ANT VERSION 1.7.1 or later.

Macros

  • Clean Module - Deletes all build artifacts for a specified module.
  • Compile Module - Compiles the main source tree and test source tree for a specified module (dealing with all necessary classpath configuration).
  • JAR Module - Builds a JAR file for a specified module. Includes main classes and omits test class. Sets the Class-Path and Main-Class manifest properties as required.
  • Test All Modules - Runs TestNG unit tests and generates reports using ReportNG.

Custom Tasks

  • DocBook - Generates PDF documentation from DocBook XML files using Saxon and Apache FOP.
